Bank loan credit dataset, visualization and maximization of profit¶

Exploring bank creddit data with Python programming language.

Project Overview

Minimization of risk and maximization of profit on behalf of the bank. The bank have has recorded a spate of NPAs (non-performing assets) that, among other thingscomprises of bad loans given out to the private consumers. The fiscal pressure has, therefore set the alarm bells ringing across all the managerial levels and hierarchy. The quest is simpleminimizes losses. A series of data for past customers pertaining to the loan applications has been collected.

When a bank gets a loan application, the bank must decide whether or not to go ahead with the loan process based on the applicant's profile. In this scenario, two types of risks are associated with the banks decision 
   If the applicant is a good credit risk, i.e. is likely to repay the loan, then not approving the loan to the person results in a loss of business to the bank. And,
   If the applicant is a bad credit risk, i.e. is not likely to repay the loan, then approving the loan to the person results in a financial loss to the bank.

To minimize losses from the bank's perspective, the bank requires a decision framework as to who and who should not be approved the loan

The Nordic Kredit contains data on 20 variables and the classification whether an applicant is considered a Good or a Bad credit risk for 1000 loan applicants. A predictive model developed on this data is expected to provide bank manager guidance for making a decision whether to approve a loan to a prospective applicant based on his/her profiles.

Model Building¶

We will be using Logistic Regression and Decision tree classifier

Logistic Regression ==> It is a technique of supervised learning. Logistic regression is a statistical model that in its basic form uses a logistic function to model a binary dependent variable, although many more complex extensions exist. In regression analysis, logistic regression (or logit regression) is estimating the parameters of a logistic model (a form of binary regression).This justifies the name logistic regression. Data is fit into linear regression model, which then be acted upon by a logistic function predicting the target categorical dependent variable. We will be using Binary Logistic Regression here as there are only 2 classes in which we have to classify.

This is the cost function of logisitc regression which is minimized.

Decision Tree ==>Decision tree learning is a method commonly used in Machine learinng. It is also a method for supervised learning. The goal is to create a model that predicts the value of a target variable based on several input variables. It is mainly used for classification though certain regressor variants of it also exist

An example of a simple decision tree.

A tree is built by splitting the source set, constituting the root node of the tree, into subsets - which constitute the successor children. The splitting is based on a set of splitting rules based on classification features. This process is repeated on each derived subset in a recursive manner called recursive partitioning. The recursion is completed when the subset at a node has all the same values of the target variable, or when splitting no longer adds value to the predictions.
